Privacy Policy / 隐私政策

Last updated / 最后更新：2025-04-11

Overview / 概述

Web2MM is an open-source browser extension that summarizes webpage content into interactive mind maps. We are committed to protecting your privacy.

Web2MM 是一款开源浏览器扩展，将网页内容总结为交互式思维导图。我们致力于保护您的隐私。

Data Collection / 数据收集

We do not collect, store, or transmit any personal data.

我们不收集、存储或传输任何个人数据。

  • No user tracking or analytics / 不追踪用户行为
  • No cookies / 不使用 Cookie
  • No personal information collected / 不收集个人信息
  • No browsing history stored / 不存储浏览历史

Data Storage / 数据存储

All data is stored locally on your device using chrome.storage.local:

所有数据通过 chrome.storage.local 存储在您的设备本地：

  • Settings: Your API key, platform selection, and model preference / 设置：您的 API Key、平台选择和模型偏好
  • Cache: AI response cache (24h TTL) to avoid redundant API calls / 缓存：AI 响应缓存（24 小时过期），避免重复调用

Neither settings nor cache are transmitted to any third-party servers.

设置和缓存均不会传输到任何第三方服务器。

Third-Party Services / 第三方服务

Web2MM sends webpage text content to AI providers for summarization. This only happens when you click the "Summarize" button:

Web2MM 将网页文本内容发送给 AI 提供商进行总结。此操作仅在您点击"总结"按钮时发生：

  • SiliconFlow (siliconflow.cn) / 国内 AI 平台
  • OpenRouter (openrouter.ai) / 海外 AI 平台

Only the extracted text content of the webpage is sent. No other data is transmitted.

仅发送提取的网页文本内容，不传输其他任何数据。

Open Source / 开源

Web2MM is fully open source. You can review all code at github.com/zhenda-hub/web2mm.

Web2MM 完全开源，您可以在 github.com/zhenda-hub/web2mm 审查所有代码。

Contact / 联系方式

If you have questions about this privacy policy, please open an issue on GitHub Issues.

如有隐私政策相关问题，请在 GitHub Issues 提交。
